На главную

Git

  • git init - инициализация нового Git-репозитория.
  • git status - показывает состояние файлов в репозитории.
  • git add . - добавляет все изменения в staging area.
  • git commit -m "message" - сохраняет изменения с сообщением.
  • git push - отправляет изменения в удалённый репозиторий.
